# Task Manager (Todo App)

## 1. プロジェクト概要
日常のタスク管理を効率化し、達成感を視覚化することを目的としたWebアプリケーションです。
単なるメモ帳ではなく、「データの正規化」や「ユーザーのUX（利用体験）」にこだわって開発しました。

## 2. 使用技術 (Tech Stack)
* **Frontend**: HTML5, CSS3 (Vanilla JS)
* **Backend**: Node.js, Express
* **Database**: MySQL
* **Authentication**: express-session, bcrypt
* **Other**: Web Notification API (リマインド機能)

## 3. 主要機能
* **ユーザー認証**: パスワードハッシュ化によるセキュアなログイン機能
* **タスク管理 (CRUD)**: 追加、編集、削除、ステータス切り替え
* **カテゴリ管理**: ユーザー独自のカテゴリ作成と紐付け（正規化の実践）
* **フィルタリング**: すべて/未完了/完了のタブ切り替え表示
* **進捗可視化**: 全タスクに対する完了率をプログレスバーで表示
* **レスポンシブデザイン**: PCおよびスマートフォン環境への対応

## 4. 技術的なこだわり
* **データベースの正規化**: 「ユーザー」「タスク」「カテゴリ」を別テーブルで管理し、JOINを用いて効率的にデータを結合しています。
* **セキュリティ**: パスワードをハッシュ化して保存し、セッション管理によるアクセス制御を実装しています。
